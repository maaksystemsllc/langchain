# Everly AI

>[Everly AI](https://everlyai.xyz/) allows you to run your ML models at scale in the cloud. 
> It also provides API access to [several LLM models](https://docs.everlyai.xyz/tutorials/model_serving).


## Installation and Setup

You have to create a new token. Please, see [instructions](https://everlyai.xyz).
