>`GigaChat` supports Russian and English languages.


## Installation and setup

- Install the `gigachat` integration package.

  ```bash
  pip install gigachat
  ```

- Get an API key by following [instructions here](https://developers.sber.ru/docs/ru/gigachat/api/reference/rest/gigachat-api).
- Authenticate by setting the `GIGACHAT_API_PERS` environment variable.
