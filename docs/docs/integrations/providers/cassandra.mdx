# Cassandra

> [Apache CassandraÂ®](https://cassandra.apache.org/) is a NoSQL, row-oriented, highly scalable and highly available database.
> Starting with version 5.0, the database ships with [vector search capabilities](https://cassandra.apache.org/doc/trunk/cassandra/vector-search/overview.html).

The integrations outlined in this page can be used with `Cassandra` as well as other CQL-compatible databases, 
i.e. those using the `Cassandra Query Language` protocol.

## Database-bound prompt templates

```python
from langchain_community.prompts import CassandraReaderPromptTemplate

my_template = CassandraReaderPromptTemplate(
  template="Greet {user_name} (aka {short_name}) in a {adj} way.",
  field_mapper={
    "user_name": ("demo_users", "user_name"),
    "short_name": ("demo_users", "short_name"),
  },
)
# table 'demo_users' has 'user_id' as primary key, so:
prompt = my_template.format(user_id="john_doe", adj="lackadaisical")
```

Learn mode in the [example notebook](/docs/modules/model_io/prompts/database_bound_prompt_templates).
