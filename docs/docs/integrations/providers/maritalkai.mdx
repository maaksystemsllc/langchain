# MariTalk AI

>[MariTalk](https://github.com/maritaca-ai/maritalk-api) is an assistant developed by the Brazilian company [Maritaca AI](https://www.maritaca.ai/). 
> `MariTalk` is based on language models that have been specially trained to understand `Portuguese` well.


## Installation and setup

See the [instructions here](https://maritaca-ai.github.io/maritalk-api/maritalk.html#instalacao).


## Chat models

See a [usage example](/docs/integrations/chat/maritalk).

```python
from langchain_community.chat_models import ChatMaritalk
```
