# Dappier AI

>[Dapier AI](https://platform.dappier.com) is a platform that provides pre-trained, 
> real-time LLM-ready data models and RAG APIs.
>
> `Daiper AI` provides [an example](https://docs.dappier.com/real-time-data-langchain) how to integrate with `LangChain`.


## Installation and Setup

You have to create a new token. Please, see [instructions](https://docs.dappier.com/introduction).


## Chat models

See a [usage example](/docs/integrations/chat/dappier).

from langchain_community.chat_models.dappier import ChatDappierAI
