# Jina

>[Jina AI](https://jina.ai/about-us) is a multimodal AI company, focusing on prompt and embedding techniques. 
> Company provides businesses and developers with innovative platforms to harness the power 
> of AI for value creation and cost savings.


## Installation and Setup

Get a `Jina AI API` token from [here](https://jina.ai/embeddings/) and set it as an environment variable (`JINA_API_TOKEN`)

## Chat models

### Jina Chat

See a [usage example](/docs/integrations/chat/jinachat).

```python
from langchain_community.chat_models import JinaChat
```

## Embedding models

You can check the list of available embedding models from [here](https://jina.ai/embeddings/)

### Jina Embeddings

See a [usage example](/docs/integrations/text_embedding/jina).

```python
from langchain_community.embeddings import JinaEmbeddings
```
